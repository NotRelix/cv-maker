## Installation
1. Clone the repository
 
    ```bash
    git clone git@github.com:NotRelix/cv-maker.git
    ```
2. Navigate to directory
   
    ```bash
    cd cv-maker
    ```
3. Install dependencies
   
    ```bash
    npm install
    ```

4. Run project
   
    ```bash
    npm run dev
    ```
